# theLab Boozang

## Description
A platform for learning  test automation and the Boozang tool. 

![Screenshot](/src/img/screenshot.jpg?raw=true "Screenshot")

## Installation
To install

```npm install```

To run

```npm start```

## To build
```npm run build``` 
Then use the dist folder

## Built With
* React.js
* Scss
* Bootstrap 4
